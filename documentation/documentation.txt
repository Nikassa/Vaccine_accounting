Веб-сервис “Учет прививок”
 

Необходимо спроектировать и разработать REST сервис «Учет прививок». Основная цель которого стандартные операции работы с пациентами и учет прививок.

Сущности:

Пациент

Фамилия [строка];
Имя [строка];
Отчество [строка];
Дата рождения [дата];
Пол [строка];
СНИЛС [строка] - необходимо проверить формат и контрольную сумму.

Прививка

Препарат [string] - выбор из списка (Эджерикс, Вианвак, АКДС, БЦЖ);
Наличие согласия прививку [boolean];
Дата проведения [date], дата вводится по маске дд.мм.гггг.


Необходимо реализовать валидацию данных по форматам полей и признаку обязательности. Кроме того  должны правильно обрабатываться ошибки отсутствия данных в запросе или в БД. Для получения списка прививок реализовать постраничный вывод.


Должны быть реализованы следующие методы:

Пациент:

Добавление;
Получение списка;
Редактирование;
Удаление пациента;
Получение пациента и списка прививок по идентификатору.

Прививка:

Добавление;
Получение по идентификатору;
Редактирование;
Удаление.

К результату добавить коллекцию запросов для Postman (https://www.postman.com/)

Требуемый инструментарий:
JAVA 8;
База данных на свой выбор;
Hibernate;
